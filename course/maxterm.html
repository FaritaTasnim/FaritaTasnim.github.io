Maximum Term Method


In statistical physics, we are often concerned with calculating logarithms of sums. In particular, we are interested in the values of the logarithm of a sum for which the number of terms in the sum goes to infinity. This tends to correspond with taking the limit that the number of systems in our ensemble goes to infinity.

In this infinite-term limit, we will often replace the logarithm of the sum with the logarithm of the term in the sum with the greatest value, called the maximum term.

each term in the sum grows exponentially with the number of terms in the sum *

Yes, there must be a relationship between the number of terms in the sum and the value of each term in the sum. If there weren't, then there would be no consistently predictable relationship between the parameters of the sum and its maximum term. Additionally, we need it to be the case that each term in the sum itself grows at least exponentially as the number of terms increases. If this were not the case, then a single term would not be able to vastly outweigh the other terms in the sum. Instead, a number of terms would hold the bulk of the weight of the sum. This number would grow as the number of terms in the sum grows.

--- Q --- Let $x$ be some constant positive real number. To which of the following logarithms of sums do you think could we safely apply the maximum term method to?

$\sum_{i=0}^m x$

$\sum_{i=0}^m i$

$\sum_{i=0}^m ix$

$\sum_{i=0}^m x^i$ *

$\sum_{i=0}^m \frac{m!x^i}{i!(m-i)!}$ *

As an example, consider the following function: $$S = \sum_{i=0}^m t_i = \sum_{i=0}^m \frac{m!*x^i}{i!(m-i)!}$$ Say that $m$ is on the order of $10^20$ and $x$ is on the order of $1$. (Notice how this sum resembles $\sum_n \Omega(n)$ from our ensemble of Grogus-in-boxes.) Let's see what happens when we consider the value of the logarithm of this sum as a function of its maximum-valued term. Do you know of a way to approximate a function around particular value of one of its parameters? It involves using how the value of the function varies as you deviate from that particular value of the parameter. Recall that this is called the 'Taylor series approximation' - a series of first-order, second-order, etc. deviations of the function in terms of first-order, second-order, etc. derivatives of the function with respect to that parameter.

Let's apply this to a function that equals the logarithm of the $i$th term in our sum, $\ln t_{i}$, expanded around $i = i^*$, the maximum term. $$\ln t_{i} \approx \ln t_{i^*} - (i - i^*) \frac{\partial \ln t_i}{\partial i}|_{i = i^*} + \frac{1}{2} (i - i^*)^2 \frac{\partial^2 \ln t_i}{\partial i^2}|_{i = i^*} \ldots $$ Remember that at the maximum value for a function with respect to a given parameter, the first derivative of the function with respect to that parameter equals $0$. This is because the value of the parameter which gives the maximum serves as a turning point. If you increase the parameter a little bit, the function decreases. o a little to the right, the derivative is negative. Similarly, if you decrease the parameter a little bit, the function also decreases. So also a little to the left, the derivative is positive. In order for this to be true, the only choice for the function at the maximizing value of the parameter is to be "flat", i.e., derivative of 0. So that means that $$\frac{\partial \ln t_i}{\partial i}|_{i = i^*} = 0 $$ so $$- \ln i^* + \ln (m - i^*) + \ln x = 0 $$ which gives that at the maximum term, $$x = \frac{i^*}{m - i^*}$$ which is equivalent to $$i^* = \frac{x*m}{1 + x}$$

We can also simplify the Taylor series approximation as: $$\ln t_{i} \approx \ln t_{i^*} + \frac{1}{2} (i - i^*)^2 \frac{\partial^2 ln t_i}{\partial i^2}|_{i = i^*} \ldots$$ $$\approx \ln t_{i^*} + \frac{1}{2} (i - i^*)^2 (- \frac{1}{i^*} - \frac{1}{m-i^*}) \ldots$$ $$\approx \ln t_{i^*} - \frac{ (i - i^*)^2 (1+x)^2}{2xM} \ldots$$ Ignoring the higher order terms and solving for t_i, we can write $t_i$ as a distribution $$t_i = t_{i^*}e^{\frac{(i - i^*)^2}{2 \sigma^2}}$$ with mean $t_{i^*}$ and standard deviation $\sigma = \frac{\sqrt{xm}}{1+x}$. This distribution looks like a Gaussian curve (AKA bell curve).

Yes - the ratio of the standard deviation to the mean provides a measure of how much the distribution of a variable typically varies with respect to its mean value. The value of this measure in terms of the variables x and m is $$\frac{\sigma}{i^*} = \frac{\sqrt{xm}}{xm} = (xm)^{-1/2}$$
